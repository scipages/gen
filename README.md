> visit the main [SciPages repository](https://github.com/scipages/scipages).

## Developer Notes

Make sure that you have Node.js and NPM installed on your machine. Preferably >= 16.x.

#### Build Setup

```bash
# Install the dependencies
npm install

# Build the app
npm run build

# Watch for file changes
npm run watch

# Lint the files
npm run lint

# Run the app
./bin/scipages
```
